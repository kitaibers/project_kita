Инфа о проекте
Я создал файл с названием .gitignore , который игнорирует файлы например если в них изменения и мы делаем гит статус, 
то они не отображаются чтоб нам просто не мешали. Но еще можно добавлять  папки, только надо ставить "/" перед названием.
Когда мы хотим сделать окончательную версию(комит), то пишем git commit -m "что то пишем например мы сделали то-то" и фиксируем опредленную версию.
git branch name_branch - создал новую ветку с каким то именем
git branch checkout name_branch - переключился на какую то ветку
git clone - клонировать проект с гита
